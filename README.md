# 🍄 CU Mushroom Coffee

A modern mushroom coffee brand website built with Next.js.

## 🌐 Live Demo

https://new-cu-mushroom-coffee.vercel.app

## 📖 Project Introduction

CU Mushroom Coffee is a modern landing page designed for a functional mushroom coffee brand.

The project focuses on:

- Modern UI Design
- Responsive Layout
- Smooth Animation
- Brand Storytelling
- Product Showcase

This project is built as an open-source learning project.
