# CLUB UNDEFINED — 开发需求文档

## 项目概述

CLUB UNDEFINED 是一个高端极简品牌网站，面向长期高压工作、对普通咖啡易焦虑的人群。品牌核心理念："在高压世界里，更稳定地活着。"

产品：低因冻干咖啡 + L-茶氨酸 + 猴头菇。

## 当前阶段目标

- 不是卖货
- 建立品牌可信度
- 建立情绪共鸣
- 收集种子用户
- 获取 waitlist 留资

## 网站 KPI

- 用户停留时间
- 留资率

## 功能需求

### v1.0（当前）

1. 单页品牌站
2. 5屏内容：Hero / Problem / Solution / ProductShowcase / Waitlist
3. Email waitlist 表单（前端日志 → 后续接后端）
4. 暗色模式
5. 慢呼吸感动效

### 后续迭代

1. Waitlist API 接入（Resend / Mailchimp）
2. 产品实拍图替换 CSS 占位
3. 用户行为分析（停留时间、滚动深度）
4. 多语言支持
